Recursos ICGC en obert

Benvinguts i benvingudes als recursos ICGC en obert Aquí trobareu el recull d'eines, estils i tessel·les, codi i exemples basats en dades ICGC.
Tots els repositoris de l'ICGC s'estan desenvolupant i mantenint per donar suport a les iniciatives de geoinformació i cartografia a Catalunya per part de l'Institut Cartogràfic i Geològic de Catalunya (ICGC).
Podeu visitar la nostra pàgina web per obtenir més informació.

Repositoris
  • MapICGC GL JS
    La biblioteca MapICGC GL JS, creada com a envoltori de MapLibreGL, ofereix noves funcions específiques en un entorn ICGC. Simplifica el desenvolupament integrant característiques com tessel·les vectorials i imatges satel·litals, i incorpora de manera immediata estils, terrenys i dades ICGC.
    Documentació MapICGC GL JS
    Biblioteca MapICGC GL JS
    Visor MapICGC GL JS

    Plantilles MapICGC GL JS
    Vanilla JS
    React
    Svelte
  • API Territorial
    Interfície de Programació d'Aplicacions que integra diverses fonts de dades geogràfiques per proporcionar informació precisa del territori a partir de coordenades.
    Informació
    Documentació API Territorial
  • Geocoder
    El Geocodificador ICGC permet obtenir coordenades d'un lloc a Catalunya a partir de l'adreça postal o topònim. També permet geocodificació inversa.
    Documentació Geocoder
    Visor Geocoder
  • Apps ICGC
    Catalunya Offline
    Aplicació mòbil per explorar el territori, marcar punts d'interès i enregistrar rutes amb GPS, fins i tot sense cobertura.
    Geo-SIV
    Programari per l'avaluació preliminar de Sistemes d'Intercanvi geotèrmic en circuit Vertical tancat.
  • Plugins ICGC
    Qgis Plugin
    Plugin d'accés a dades obertes de l'ICGC. Proporciona cerques de geocodificació.
